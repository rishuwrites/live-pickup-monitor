# Live Pickup Monitor

![Live Monitor Screenshot](assets/Screenshot.png)

This project helps users to monitor live pickups and provides real-time updates.

## Features

- Real-time statistics
- Live tracking of pickups
- Notification system for updates

## Installation

To install this project, clone the repository and run the following command:

```bash
npm install
```

## Usage

Run the application using:

```bash
npm start
```
